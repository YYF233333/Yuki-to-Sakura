脚本归类：
*character: 用于存放角色对象及其属性
*cg: 存放背景和cg及其属性

*chapterX: 用于存放各章脚本
*gui: GUI相关
*options: 自定义游戏设置

3. 合理使用空行来切分文本
4. 非文本部分参照代码格式要求及一般编码规范
